用hexo在github上搭建静态博客

ljm11111的博客
2020-05-23

1、需要安装的软件

  • node.js

  • git fro windows

  • hexo

  • sublime text

2、搭建的步骤

1、创建一个github域名，这个域名就是你以后博客的网址。

2、设置一个ssh key，这样的话才能把把本地的内容通过github账户上传到自己的博客上面。

3、通过sublime text编写本地的网站的一些具体信息。

4、把本地编写的内容通过hexo上传到github的博客当中

5、最后你还能在hexo的官网找喜欢的皮肤装上

3、遇到的一个问题

在我使用npm install命令的时候出现”Unexpected end of JSON
input while parsing near”的错误，百度过后用
npm cache clean --force这个命令解决了。
